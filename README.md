Melodict is a web-based data analytics dashboard designed to bridge the gap between digital music consumption, user psychology, and daily productivity. By interfacing with streaming data concepts (modeled after the Spotify API), Melodict transforms raw audio features into actionable lifestyle and performance insights.

The project rejects traditional, uninspired CRUD layouts in favor of a high-impact, Neo-Brutalist design system that prioritizes hard data visualization, fluid responsiveness, and low-latency client-side data persistence.

Key Feature Breakdown (An Analyst's Perspective)
The Data Analyst View (Advanced Visualization): Built an interactive analytics workspace using Chart.js to render complex multi-variable data layouts, including Radar Charts for sonic profile mapping and multi-axis Line Graphs to identify cognitive performance thresholds.

Biometric & Rhythmic Correlation Engine: Developed a custom data-logging matrix on the Wellness Hub. It captures user mood states, habit completion ratios, and physical heart rates (BPM), dynamically computing a weighted Daily Wellness Score mapped against audio metadata via Scatter Plots.

Asynchronous State & Session Management: Engineered a high-precision, timestamp-based session stopwatch using absolute Unix Timestamps. The architecture utilizes the native browser Storage Event API to achieve real-time, cross-tab asynchronous data synchronization without draining device battery through polling loops.

Symmetric Engineering & DRY Architecture: Implemented a highly maintainable Symmetric Naming Convention (domain-driven isolation of HTML, CSS, and JS components) and eliminated duplicate scripts across layout components to ensure production-grade codebase scalability.

Technical Stack & Architectural Design
Data Architecture: LocalStorage API, JSON Serialization, Asynchronous Event Delegation.

UI/UX Framework: Semantic HTML5, CSS Grid (Fluid auto-fit & minmax layouts), Responsive Typography (clamp()), Bootstrap 5.

Data Graphics Engine: Chart.js (Hardware-accelerated raster canvas rendering).

Design Philosophy: Neo-Brutalism (Zero-blur high-contrast solids, rigid typography, raw borders).
